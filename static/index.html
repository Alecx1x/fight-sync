      ⬆ Upload to YouTube
      Uploads this video with the title, description & tags above. Unverified Google projects may force the video to Private until verified — that's a YouTube policy.
      One-time setup to enable upload:
      1. In Google Cloud, create a project & enable YouTube Data API v3.
      2. Create an OAuth client ID (type Desktop app); save the JSON as client_secret.json in the fight-sync folder.
      3. Run once in the terminal: .venv\Scripts\python youtube_auth.py
